Welcome to the Url Shortener

  • This is a microservice that will take a url and generate a shortened version of that url

  • To use it enter a valid url in the form of localhost:3000/new/http://www.example.com

  • When you search for that url, a link will be generated for both the full version of the url and the shortened version

  • If you have a shortened link, you can enter the text for that code in the form localhost:3000/eXamPl to return the original url for that link
